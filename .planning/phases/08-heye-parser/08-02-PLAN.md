heye.kr 검색 결과를 가져오는 API 엔드포인트를 생성한다. 페이지네이션 지원.

구현 사항:
1. GET /api/search/heye?q=검색어&page=1
2. heye.kr 검색 URL 호출:
   `https://www.heye.kr/board/index.html?id=idol&smode=both&skey=${encodeURIComponent(query)}&page=${page}`
3. cheerio로 HTML 파싱:
   - 게시글 목록에서 각 항목의 링크(no 파라미터), 제목, 썸네일 추출
   - 결과 패턴: `./index.html?id=idol&no=12345` → 절대 URL로 변환
   - 총 페이지 수 추출 (페이지네이션 링크에서)
4. 각 결과를 `{ url, title, thumbnailUrl, author }` 형태로 반환
5. 응답에 `{ results, totalPages, currentPage }` 포함

에러 처리:
- 네트워크 오류 시 500 + error message
- 결과 없을 시 빈 배열 + totalPages: 0

반환 타입:
```typescript
interface HeyeSearchResponse {
  results: {
    url: string      // https://www.heye.kr/board/index.html?id=idol&no=12345
    title: string
    thumbnailUrl: string | null
    author: string
  }[]
  totalPages: number
  currentPage: number
}
```

외부 검색 모달에 heye.kr 플랫폼과 페이지네이션 UI를 추가한다.

1. `src/lib/parsers/index.ts`:
   - Platform 타입에 'heye' 추가 (08-01에서 이미 추가됨, 확인만)

2. `src/components/ExternalSearch.tsx`:
   - Platform 타입에 'heye' 추가
   - 플랫폼 선택 버튼 추가 (오렌지색 테마)
   - heye 검색용 상태 추가:
     ```typescript
     const [heyePage, setHeyePage] = useState(1)
     const [heyeTotalPages, setHeyeTotalPages] = useState(0)
     ```
   - searchHeye 함수 추가 (페이지 파라미터 지원):
     ```typescript
     const searchHeye = async (searchQuery: string, page: number = 1): Promise<EnrichedResult[]> => {
       const response = await fetch(`/api/search/heye?q=${encodeURIComponent(searchQuery)}&page=${page}`)
       const data = await response.json()
       setHeyeTotalPages(data.totalPages)
       setHeyePage(data.currentPage)
       return data.results.map(...)
     }
     ```
   - handleSearch에서 platform === 'heye' 케이스 추가
   - heye 플랫폼일 때 페이지네이션 UI 렌더링:
     ```tsx
     {platform === 'heye' && heyeTotalPages > 1 && (
       <div className="flex justify-center gap-2">
         <button onClick={() => searchHeye(query, heyePage - 1)} disabled={heyePage <= 1}>이전</button>
         <span>{heyePage} / {heyeTotalPages}</span>
         <button onClick={() => searchHeye(query, heyePage + 1)} disabled={heyePage >= heyeTotalPages}>다음</button>
       </div>
     )}
     ```
   - 결과 카드에 heye 플랫폼 뱃지 스타일 추가 (오렌지색)

UI 스타일:
- 버튼: bg-orange-100 dark:bg-orange-900/50 text-orange-700 dark:text-orange-300
- 뱃지: 같은 오렌지 테마
- 페이지네이션: 결과 목록 하단에 이전/다음 버튼 + 현재 페이지 표시
